Zeqa Terminal

A terminal-based program for Minecraft server utilities, focused on the Zeqa Network. Simplify server management and player tracking with powerful tools.

Features:

  • Get online players from Minecraft servers
  • Fetch server details including MOTD, plugins, and version
  • Access Bedrock server software download links
  • Zeqa Network utilities like player tracking and lobby status
Visit Zeqa Terminal

BDS-Manager

A lightweight and efficient tool for managing Bedrock Dedicated Servers (BDS), featuring built-in Playit protocol for seamless tunneling.

Features:

  • Simplified server setup and management
  • Automatic tunneling with Playit protocol
  • Customizable configurations
  • Efficient resource management
